# Global Process (Golden Image)

Evaluate → Converge → Implement → Verify

## Evaluate
- Inventory current governance wrapper files
- Inventory AI artifacts and violations (tracked scratch, scattered docs)

## Converge
- If existing docs are complete: map them into the standard structure
- Otherwise: ask minimal questions and propose defaults

## Implement (batched)
- Batch 1: wrappers + `.ai` skeleton + ledger
- Batch 2: bundle installation
- Batch 3: rehome/dedupe

## Verify
- No tracked `.ai/_scratch/`
- manifest tier-1 entrypoints correct
- ledger updated
