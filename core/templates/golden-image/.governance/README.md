# Repository Governance Wrapper

This directory is the **governance entrypoint** for this repository.

- `.governance/ai/` is a Git submodule containing shared governance rules and routing.
- `.governance/manifest.json` is the discovery contract for AI context loading.
- `.governance-local/overrides.yaml` holds repo-specific toggles and integrations.
